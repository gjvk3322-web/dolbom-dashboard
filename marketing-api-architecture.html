🗺 단계별 구현 로드맵

1
AWS Flask API 서버 세팅
⏱ 1~2시간

기존 AWS EC2에 Flask 서버를 추가로 띄워서 marketing.html의 백엔드로 사용합니다.

  • Flask + gunicorn 설치
  • CORS 설정 (my.dolbommat.com 허용)
  • API 키를 .env 파일로 안전 관리
  • 기본 엔드포인트 구성 (/api/naver, /api/ai, /api/notify)
# AWS 서버에서 실행 # ~/돌봄매트/marketing_api/app.py from flask import Flask, jsonify, request from flask_cors import CORS import os app = Flask(__name__) CORS(app, origins=["https://my.dolbommat.com"]) @app.route('/api/health') def health(): return jsonify({"status": "ok"}) @app.route('/api/naver/keywords', methods=['POST']) def naver_keywords(): # 네이버 검색광고 API 호출 ... @app.route('/api/ai/generate', methods=['POST']) def ai_generate(): # Claude API 호출하여 콘텐츠 생성 ... @app.route('/api/notify/telegram', methods=['POST']) def notify_telegram(): # 텔레그램 알림 발송 ...
2
네이버 검색광고 API 연동 (데이터 자동 수집)
⏱ 2~3시간

이미 API 키를 보유하고 계시니 바로 연동 가능합니다.

  • 경쟁사 20개 브랜드 월간검색량 자동 수집
  • 돌봄매트 검색광고 노출/클릭/비용/전환 일별 데이터
  • 시장조사 노드 & 네이버광고 노드에 실시간 데이터 표시
  • 매일 09:00 cron으로 자동 수집 → 구글시트 저장
# /api/naver/keywords → 경쟁사 검색량 조회 # /api/naver/ad-stats → 광고 성과 데이터 조회 # 워크플로우 노드에서 버튼 클릭 시: # marketing.html → fetch('https://3.35.253.214:5000/api/naver/keywords') # → 노드 카드에 "돌봄매트: 12,400 / 토리매트: 8,200 / ..." 자동 표시
3
AI 콘텐츠 자동 생성 (Claude API)
⏱ 2~3시간

콘텐츠 제작 노드에서 버튼 하나로 AI가 마케팅 콘텐츠를 생성합니다.

  • 블로그 포스팅 원고 자동 생성 (키워드 + 시공사진 기반)
  • 인스타그램 캡션 + 해시태그 자동 생성
  • 네이버 광고 소재 문구 A/B 테스트 자동 생성
  • 맘카페 공구 모집글 템플릿 자동 생성
# 콘텐츠 노드에서 "🤖 AI 생성" 버튼 클릭 시: # → AWS 서버가 Claude API 호출 # → 프롬프트: "돌봄매트 1000매트 22T 층간소음매트 블로그 포스팅을 작성해줘. # 타겟: 아파트 입주 예정 30대 부부, 키워드: 층간소음매트 추천..." # → 생성된 원고가 노드 메모에 자동 저장
4
텔레그램 알림 자동화
⏱ 1시간

기존 텔레그램 봇을 활용해서 마케팅 리마인더를 자동화합니다.

  • 캠페인 마감 D-1 자동 알림 (매일 09:00 체크)
  • 주간 마케팅 성과 리포트 자동 발송 (월요일 오전)
  • 광고 예산 소진율 경고 (80% 도달 시)
  • 새 전환(계약) 발생 시 실시간 알림
# 텔레그램 알림 예시: # 📢 [마케팅 리마인더] # 내일 마감: "블로그 콘텐츠 3건 발행" # 네이버 광고 예산 소진율: 82% (12일 남음) # 이번 주 성과: 노출 18,500 / 클릭 520 / 전환 6건
5
구글시트 연동 (데이터 영구 저장)
⏱ 2시간

현재 localStorage에 저장되는 워크플로우 데이터를 구글시트로 옮겨서 어디서든 접속 가능하게 합니다.

  • 마케팅_워크플로우 시트 새로 생성
  • 노드 데이터 / 연결 데이터 / 성과 데이터 시트 분리
  • Apps Script 경유로 읽기/쓰기
  • 자동 저장 + 실시간 동기화
