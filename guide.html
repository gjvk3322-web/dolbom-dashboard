검수 체크리스트

하나씩 확인 후 체크해 주세요

1. 작업 상태 확인
방문, 손끼임방지 확인
방문 앞 여유 공간 및 손끼임 방지 처리 상태
화장실, 펜트리 단면처리 확인
단면 마감이 깔끔하게 처리되었는지 확인
벽 휜 경우 틈 발생 안내
벽이 휜 부분에 약간의 틈이 생길 수 있음을 설명
매트 들뜸 확인 및 밀착시공 설명
들뜸 없이 바닥에 밀착되었는지 확인
잘린 단면 황변 설명
시간이 지나면 잘린 단면이 자연적으로 황변될 수 있음 안내
펜트리 문 열림 확인
매트 시공 후 펜트리 문이 정상적으로 열리는지 확인
냉장고 문 열림 확인
매트 높이로 인해 냉장고 문이 걸리지 않는지 확인
중문 슬라이드 걸림 확인
슬라이드 중문이 원활하게 작동하는지 확인
2. 가구 원위치 상태 확인
쇼파 받침 확인
쇼파가 안정적으로 놓여있는지 확인
TV 연결 상태 확인
TV 전원 및 연결 케이블이 정상인지 확인
3. 기타 확인
압박시공 안내 완료
벽과 벽 사이보다 매트를 크게 재단하여 압박 시공하면 추후 벌어짐을 방지할 수 있음을 설명
작업자 짐 챙김 확인
공구, 자재 등 작업자 짐을 모두 가져갔는지 확인
