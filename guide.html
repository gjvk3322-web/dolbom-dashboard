1. 작업 상태 확인

화장실, 펜트리 단면처리 확인
단면 마감이 깔끔하게 처리되었는지 확인
벽 휜 경우 틈 발생 안내

들뜸 없이 바닥에 밀착되었는지 확인

펜트리 문 열림 확인
매트 시공 후 펜트리 문이 정상적으로 열리는지 확인
냉장고 문 열림 확인
매트 높이로 인해 냉장고 문이 걸리지 않는지 확인
중문 슬라이드 걸림 확인
슬라이드 중문이 원활하게 작동하는지 확인
